Cordia File Explorer

심플함에 집중한 싱글 패널형 파일 관리 도구. 가볍고 빠른 파일 열람과 iOS 기기 마운트 지원이 특징입니다.

다운로드

Newton Commander

프로들을 위한 듀얼 패널 전용 네이티브 macOS 파일 매니저. 직관적인 단축키와 뛰어난 탐색 효율을 제공합니다.

다운로드

KeyMap

업무 속도를 가속화하는 시스템 전역 단축키 및 키 매핑 도구. 10개 언어 팩을 완벽하게 지원합니다.

다운로드

Quick Folder

사용자가 원하는 곳으로 언제든 신속하게 파일을 이동할 수 있게 만들어 주는 스위칭 폴더 팝업 어시스턴트.

다운로드

scapC

강력한 화면 캡처 및 즉각 공유를 위한 유틸리티. 원하는 영역을 쉽고 직관적으로 획득하세요.

다운로드

BluePass

복잡한 정보를 안전하게 관리해 주는 보안 특화 인터페이스 도구. 당신의 소중한 정보를 안전하게 보호합니다.

다운로드

EPGMN

Mac 시스템을 더욱 강력하게 다룰 수 있도록 돕는 고급 제어 유틸리티입니다.
